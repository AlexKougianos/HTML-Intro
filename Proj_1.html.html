Basic tags

Type <tag_name> Text </tag_name>
  • h1-6 for Headers (1 being the biggest)
  • p for paragraph
  • strong for bold
  • em for italic
  • u for underline
  • ol for ordered list
  • ul for unordered list
  • li for each list item (Each item can have a sub-list)

  • br for break/end line

Other tags

  • img src = "Image_Source" to insert an image. The Image_Source can be a URL or the name of an image in the folder of the html file. This tag doesn't need a closing tag.
  • a href = "URL" to insert links in the file.
    • google.com
    • youtube.com
